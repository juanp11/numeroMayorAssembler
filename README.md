Laboratorio #1: Simulación y optimización de un programa en un procesador escalar segmentado
•	Ejercicio 1: número mayor.
Se solicitan que se ingresen tres números por teclado, y se hace dos if, el primer para compara el mayar entre el primer y segundo número, luego se ejecuta un segundo if para validar el número mayor de la validación anterior, y así se obtiene el numero mayor de los tres números ingresados.
